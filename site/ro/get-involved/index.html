Implicați-vă

Program la scară umană

BookWyrm este conceput pentru a rămâne mic și personal. Dacă aveți cereri de funcționalități, raporturi de buguri sau vise mărețe, ne-ar plăcea să vă alăturați conversației.

Faceți cunoștință cu întreținătorul

BookWyrm este menținut de Mouse Reeve care dirijează proiectul, gestionează comunitatea de dezvoltatori și încearcă să repare bugurile mai repede decât le introduce. Scopul primar este de a face BookWyrm un loc amabil și ospitalier.

Buguri și cereri de funcționalități

Gestionarul de buguri BookWyrm și codul sunt găzduite de GitHub — este cel mai bun loc pentru a raporta buguri și a face cereri de funcționalități.

Traduceri

Help translate BookWyrm into other languages! You can learn more about the translation process in our documentation.

Rețele sociale

Puteți urmări BookWyrm pe Mastodon și Twitter pentru a fi la curent cu noile funcționalități. Nu ezitați să menționați contul sau să trimiteți un mesaj direct dacă doriți să intrați în contact.

Chatul dezvoltatorilor

Folosim Matrix pentru chatul de dezvoltare BookWyrm — este un loc minunat pentru a pune întrebări și a obține ajutor. Se concentrează pe dezvoltarea programului BookWyrm și a funcționării instanțelor de BookWyrm, dar orice tip de discuție BookWyrm este binevenit.

BookWyrm este un program colaborativ, anti-corporație menținut de Mouse Reeve.
